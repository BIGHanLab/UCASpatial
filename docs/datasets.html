Dataset Preparation Guide

1. Data Requirements

For UCASpatial, you need:

  • Single-cell reference data with cell type annotations
  • Spatial transcriptomics data from same tissue type
  • Matching gene annotations between datasets

2. Format Conversion

Convert your data to required formats:

# R format
sc_ref <- as.Seurat(sce_object)
st_vis <- as.Seurat(st_object)

# Python format
adata_sc = sc.read_h5ad("sc_data.h5ad")
adata_st = sc.read_h5ad("st_data.h5ad")

3. Quality Control

Basic QC steps:

  • Filter low-quality cells/spots
  • Remove doublets
  • Check mitochondrial content
  • Validate cell type annotations
